== Description ==

Sweet Sharmin Preloader  will create a preloading screen for your website before all your images (including the images in CSS) are fully loaded.

It will show 3D preloading effect before loading website that looks awesome.

Specification:
Light weight only 35kb.
Easy to install
No shortcode required.

Demo link: http://sharmin.byethost7.com/

== Installation ==

1. Upload `Sweet Sharmin preloader` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now refresh browser and enjoy.
